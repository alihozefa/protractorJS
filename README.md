# Protractor
This project shows how to setup protractor on a local machine and develop scripts through eclipse.

### Install Angular JS plugin and convert to intern project
```
http://oss.opensagres.fr/angularjs-eclipse/1.2.0/
```

If you cannot find the plugin in market place :

Go to Help-> Install New Software-> copy the above URL and press Enter.. U can find the required options to install.
